# form-vue3

重温之前在金山训练营的项目,本来想优化一下之前基于vue2写的接口,以及组件啥的,完善一下之前的项目,发现之前写的确实不太好.于是想用刚学的vue3来进行新的重构,顺便完善自己之前写的不好的地方,同时学习新的vue3的知识.

---

>食用:
>
>* 1下载依赖包
>
>```node.js
>npm install
>```
>
>* 2运行后端(先找到后端根目录) [form-serve](https://github.com/2019VK/form-serve)
>
>```
>nom run serve
>```
>
>* 3运行前端
>
>```
>npm run serve 
>```
